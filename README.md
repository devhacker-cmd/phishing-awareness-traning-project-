# CODE ALPHA  phishing-awareness-traning-project-
## What is Phishing?
Definition of phishing
How phishing attacks work
## Types of Phishing Attacks
Email phishing
Spear phishing
Whaling
Smishing (SMS phishing)
Vishing (Voice phishing)
Clone phishing
## Common Signs of a Phishing Attempt
Suspicious sender addresses
Urgent or threatening messages
Fake links and attachments
Grammar and spelling mistakes
## How to Identify Phishing Emails
Verify the sender
Hover over links before clicking
Check for HTTPS and legitimate domains
## Real-World Phishing Examples
Fake banking emails
Fake delivery notifications
## Social media account verification scams
Prevention Techniques
Enable multi-factor authentication (MFA)
Use strong, unique passwords
Keep software updated
Avoid clicking unknown links
Verify requests through official channels
## What to Do If You Are Phished
Disconnect from the network if malware is suspected
Change passwords immediately
Enable MFA
Report the incident
Scan the device for malware
Phishing Awareness Quiz
Create simple questions to test users' knowledge.
